Method · what scale buys

The same question, growing up.

Prompt: "To bake bread, you first need" — asked to each model as it grew. Watch coherence arrive before, then with, correctness.

SEED · 45M
"…you can learn how to play music again. When you start playing music, you can start to learn how to play it…"

SPROUT · 150M

SAPLING · 500M

An actual recipe — then it quizzes itself and answers correctly. That's what scale buys.
